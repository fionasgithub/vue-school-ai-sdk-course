# AI Interfaces with Vue, Nuxt, and the AI SDK

## 關於此專案

這是 [Vue School](https://vueschool.io) 課程的練習專案。

### 課程資訊

- **課程名稱**：AI Interfaces with Vue, Nuxt, and the AI SDK
- **課程網址**：https://vueschool.io/courses/ai-interfaces-with-vue-nuxt-and-the-ai-sdk

### 課程簡介

本課程將帶你使用 AI SDK v6、Vue 與 Nuxt 建構 AI 應用程式。課程內容涵蓋文字與圖片生成、串流（Streaming）、結構化資料處理、工具呼叫（Tool Calling）、MCP 工具整合，以及完整的對話式聊天 UI 實作。

使用現成的 [Nuxt UI Chat template](https://github.com/nuxt-ui-templates/chat) 去串接 AI SDK。

### 技術

- [AI SDK](https://ai-sdk.dev/): v6
- AI SDK Provider - [Google Generative AI Provider](https://ai-sdk.dev/)
